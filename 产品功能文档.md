## 系统概述

### 系统简介
实验室多端Web库存管理系统是一个基于React + Flask架构的现代化库存管理平台,支持多端访问(PC端、移动端),提供灵活的表格定义、数据管理、统计分析等功能。

### 技术架构
- **前端**: React 18 + Ant Design + React Router
- **后端**: Flask + SQLAlchemy + JWT认证
- **数据库**: SQLite
- **部署**: 支持PyInstaller打包为独立可执行文件

### 核心特性
- 动态表格定义,支持自定义字段
- 灵活的数据管理,支持增删改查
- 多级权限控制(管理员、组长、组员)
- 数据导入导出,支持CSV和Excel格式
- 响应式设计,完美适配移动端
- 多语言支持(中文、英文)
- 单元格链接功能,支持链接到网页和表格
- 自增序列功能,支持前缀配置

## 用户认证与权限管理

### 1. 登录功能

#### 功能描述
用户通过用户名和密码登录系统,支持"30天内免登录"功能。

#### 页面元素
- **系统标题**: "实验室库存管理系统" / "Lab Inventory Management System"
- **用户名输入框**: 
  - 占位符: "用户名" / "Username"
  - 必填字段
  - 支持自动填充
- **密码输入框**: 
  - 占位符: "密码" / "Password"
  - 支持显示/隐藏密码切换
  - 必填字段
- **记住我复选框**: 
  - 文本: "30天内免登录" / "Remember me for 30 days"
  - 勾选后token有效期30天,否则1小时
- **登录按钮**: 
  - 文本: "登录" / "Login"
  - 点击后进行身份验证
- **重置密码链接**: 
  - 文本: "重置密码" / "Reset Password"
  - 跳转到密码重置流程

#### 交互流程
1. 用户输入用户名和密码
2. 点击"登录"按钮
3. 系统验证用户名和密码
4. 验证成功:
   - 普通登录: 生成JWT token,跳转到主页
   - 首次登录/默认密码: 提示需要重置密码,跳转到重置密码页面
5. 验证失败: 显示错误提示"用户名或密码错误"

#### 特殊逻辑
- 首次登录必须重置密码
- 使用默认密码(000000)登录必须重置密码
- Token过期后需要重新登录
- 支持密码为空的情况(特殊处理)

#### 默认账号
- **管理员**: admin / admin123
- **组长**: leader / leader123
- **组员**: member / member123
- **测试机器人**: testbot_playwright / 000000

### 2. 密码重置功能

#### 功能描述
首次登录或使用默认密码时,系统强制要求用户重置密码。

#### 页面元素
- **页面标题**: "设置新密码"
- **用户名显示**: 显示当前重置密码的用户名
- **新密码输入框**: 
  - 占位符: "请输入新密码"
  - 支持显示/隐藏密码切换
  - 最小长度: 1个字符
- **确认新密码输入框**: 
  - 占位符: "请确认新密码"
  - 支持显示/隐藏密码切换
  - 实时验证与新密码是否一致
- **提交按钮**: 
  - 文本: "提交"
  - 点击后保存新密码

#### 交互流程
1. 用户进入密码重置页面
2. 输入新密码
3. 输入确认密码
4. 系统验证两次密码是否一致
5. 验证成功:
   - 更新用户密码
   - 标记用户为非首次登录
   - 显示成功提示
   - 1.5秒后跳转到登录页面,并自动填充用户名
6. 验证失败: 显示错误提示

#### 验证规则
- 新密码不能为空
- 两次密码必须一致
- 密码强度无特殊要求(长度>=1即可)

## 表格定义功能

### 1. 表格定义页面

#### 功能描述
管理员可以自定义表格结构,包括表格名称、描述、字段配置等。

#### 页面元素
- **页面标题**: "表格定义"
- **表格名称输入框**: 
  - 占位符: "请输入表格名称"
  - 必填字段
- **表格描述输入框**: 
  - 占位符: "请输入表格描述"
  - 可选字段
- **字段列表区域**: 
  - 显示所有已定义的字段
  - 每个字段包含:字段名称、数据类型、字段长度、是否必填、下拉选项、自增配置、是否隐藏
- **添加字段按钮**: 
  - 文本: "添加字段"
  - 点击后添加新的字段行
- **保存表格按钮**: 
  - 文本: "保存表格"
  - 点击后保存表格结构

#### 字段配置选项

**字段名称**
- 必填字段
- 字符类型
- 不能重复

**数据类型**
- string: 字符串类型
- number: 数字类型
- date: 日期类型

**字段长度**
- 可选字段
- 仅对string类型有效
- 用于输入验证

**是否必填**
- 复选框
- 勾选后该字段为必填项

**下拉选项**
- 可选功能
- 配置下拉菜单的选项
- 格式: 选项1,选项2,选项3
- 勾选后字段变为下拉选择

**自增配置**
- 可选功能
- 配置自增序列
- 前缀: 自增值的前缀(如"SN-")
- 勾选后该字段自动生成唯一值

**是否隐藏**
- 复选框
- 勾选后该字段在表格中隐藏,仅在展开行中显示

#### 交互流程
1. 管理员进入表格定义页面
2. 输入表格名称和描述
3. 点击"添加字段"按钮
4. 配置字段属性
5. 可重复添加多个字段
6. 点击"保存表格"按钮
7. 系统验证:
   - 表格名称不能为空
   - 至少需要一个字段
   - 每个字段必须有名称和数据类型
8. 验证成功:
   - 保存表格结构
   - 显示成功提示
   - 跳转到已定义表格页面
9. 验证失败: 显示错误提示

#### 特殊功能
- **自增序列检查**: 在设置自增字段前,可以检查现有数据是否符合格式要求
- **字段删除**: 可以删除已添加的字段
- **字段编辑**: 可以修改已添加的字段配置

### 2. 已定义表格页面

#### 功能描述
查看所有已定义的表格,支持编辑、删除等操作。

#### 页面元素
- **页面标题**: "已定义表格"
- **表格列表**: 
  - 显示所有表格
  - 列包括: ID、表格名称、描述、创建时间、更新时间、操作
- **操作按钮**: 
  - 编辑: 修改表格结构
  - 删除: 删除表格及其所有数据
  - 管理数据: 跳转到数据管理页面
- **删除所有表格按钮**: 
  - 文本: "删除所有表格"
  - 仅admin用户可见
  - 点击后删除所有表格和数据

#### 交互流程
1. 用户进入已定义表格页面
2. 查看表格列表
3. 点击"编辑"按钮:
   - 跳转到表格定义页面
   - 加载现有表格结构
   - 可以修改字段配置
4. 点击"删除"按钮:
   - 弹出确认对话框
   - 确认后删除表格及其所有数据
5. 点击"管理数据"按钮:
   - 跳转到数据管理页面
   - 显示该表格的数据

#### 权限控制
- 管理员: 可以编辑、删除表格
- 组长/组员: 只能查看表格,不能编辑或删除

## 数据管理功能

### 1. 数据管理页面

#### 功能描述
对指定表格的数据进行增删改查操作,支持批量操作、搜索、筛选等功能。

#### 页面元素
- **页面标题**: "数据管理 - {表格名称}"
- **面包屑导航**: 
  - 首页 > 已定义表格 > 数据管理
- **操作按钮区域**: 
  - 添加数据按钮
  - 导出数据按钮
  - 导入数据按钮
  - 统计分析按钮
- **搜索框**: 
  - 占位符: "搜索..."
  - 支持全文搜索
- **筛选器**: 
  - 根据字段筛选数据
- **数据表格**: 
  - 显示表格数据
  - 支持分页
  - 支持排序
  - 支持展开行(显示隐藏字段)
- **批量操作区域**: 
  - 选中行后显示
  - 包含:取消按钮、确认删除按钮

#### 表格功能

**1. 列显示**
- 选择列: 复选框,支持单选、多选、全选
- ID列: 显示数据ID
- 创建时间列: 显示数据创建时间(只显示年月日)
- 数据列: 显示各字段数据
- 操作列: 包含编辑、删除、展开/收起按钮

**2. 行选择**
- 单选: 点击复选框
- 多选: 按住Ctrl/Cmd点击复选框
- 连续选择: 按住Shift点击复选框
- 全选: 点击表头复选框

**3. 单元格交互**
- **右键菜单**: 
  - 复制: 复制单元格内容
  - 链接到表格: 将单元格链接到其他表格的某一行
  - 链接到网页: 将单元格链接到网页
  - 取消链接: 取消已设置的链接
- **长按菜单**(移动端): 长按单元格500ms触发右键菜单
- **链接显示**: 
  - 表格链接: 蓝色下划线,点击跳转到目标表格并展开对应行
  - 网页链接: 蓝色下划线,点击在新标签页打开网页

**4. 表头交互**
- **右键菜单**: 
  - 显示字符长度: 在该列前显示字符长度列
- **字符长度列**: 
  - 显示单元格内容的字符长度
  - 浅蓝色背景
  - 可以按需显示/隐藏

**5. 展开行**
- 点击展开按钮显示隐藏字段
- 展开行以表格形式展示详细信息
- 支持在展开行中进行单元格交互

**6. 响应式设计**
- PC端: 完整显示所有列
- 移动端: 隐藏部分列,通过展开行查看
- 支持横向拖动滚动

**7. 高亮显示**
- 新添加的行会高亮显示(蓝色边框闪烁3次)
- 选中行有红色背景色

**8. 表格样式**
- 行高优化为原来的3/4
- 交替行颜色(浅灰/白色)
- 悬停效果(浅灰色背景)
- 操作按钮缩小为原来的2/3

#### 数据操作

**1. 添加数据**
- 点击"添加数据"按钮
- 弹出添加数据模态框
- 根据表格字段配置生成表单
- 自增字段自动生成,不可编辑
- 下拉字段显示下拉选择框
- 必填字段有验证
- 点击"确定"保存数据

**2. 编辑数据**
- 点击行的"编辑"按钮
- 弹出编辑数据模态框
- 加载现有数据
- 可以修改所有字段(自增字段除外)
- 点击"确定"保存数据

**3. 删除数据**
- 单条删除: 点击行的"删除"按钮
- 批量删除: 选中多行后点击"确认删除"按钮
- 弹出确认对话框
- 确认后删除数据

**4. 搜索数据**
- 在搜索框输入关键词
- 系统实时搜索所有字段
- 显示匹配的数据

**5. 筛选数据**
- 点击筛选器按钮
- 根据字段配置筛选条件
- 应用筛选条件
- 显示符合条件的数据

**6. 排序数据**
- 点击表头
- 按该列升序/降序排序
- 默认按创建时间降序

**7. 分页**
- 支持分页显示
- 可调整每页显示数量(10/20/50/100)
- 显示总记录数和当前页码

#### 权限控制
- 管理员: 所有操作
- 组长: 添加、编辑、删除、搜索、筛选、排序、分页
- 组员: 添加、编辑、搜索、筛选、排序、分页(不能删除)

### 2. 链接功能

#### 功能描述
支持将单元格内容链接到网页或其他表格的行,实现数据关联。

#### 链接到网页
1. 右键点击单元格
2. 选择"链接到网页"
3. 弹出链接输入框
4. 输入网址(支持http/https,自动添加https://前缀)
5. 点击"确定"
6. 单元格内容变为可点击链接
7. 点击链接在新标签页打开网页

#### 链接到表格
1. 右键点击单元格
2. 选择"链接到表格"
3. 弹出表格选择框
4. 选择目标表格
5. 显示目标表格数据
6. 选择目标行
7. 点击"确定"
8. 单元格内容变为可点击链接
9. 点击链接跳转到目标表格并展开对应行

#### 取消链接
1. 右键点击已链接的单元格
2. 选择"取消链接"
3. 链接被移除,单元格恢复为普通文本

#### 链接数据格式
- **网页链接**: `{ _text: "显示文本", _link: "https://example.com" }`
- **表格链接**: `{ _text: "显示文本", _link: "table:{tableId}:{rowId}" }`

### 3. 自增序列功能

#### 功能描述
支持字段自动生成唯一值,可以配置前缀。

#### 配置方式
1. 在表格定义中,勾选字段的"自增"选项
2. 输入前缀(可选)
3. 保存表格结构

#### 自增规则
- 自增值格式: 前缀 + 数字(如: SN-1, SN-2, SN-3)
- 每次添加数据时自动递增
- 从现有数据的最大值+1开始
- 支持检查现有数据是否符合格式要求

#### 检查功能
在设置自增字段前,可以检查现有数据:
1. 点击"检查自增"按钮
2. 系统验证现有数据格式
3. 返回最大值和前缀
4. 如果数据不符合格式,显示错误信息

## 报表统计功能

### 1. 报表统计页面

#### 功能描述
对表格数据进行统计分析,支持按日期或指定字段分组统计。

#### 页面元素
- **页面标题**: "报表统计 - {表格名称}"
- **面包屑导航**: 
  - 首页 > 已定义表格 > 数据管理 > 报表统计
- **统计配置区域**: 
  - 起始日期选择器
  - 结束日期选择器
  - 分组字段选择器
  - 生成报表按钮
- **统计图表**: 
  - 柱状图
  - 显示分组统计结果
- **统计表格**: 
  - 显示详细统计数据
  - 列包括: 分组项、数量

#### 交互流程
1. 用户进入报表统计页面
2. 选择起始日期和结束日期(可选)
3. 选择分组字段(可选,默认按日期分组)
4. 点击"生成报表"按钮
5. 系统查询数据并统计
6. 显示统计图表和表格
7. 可以导出统计结果

#### 统计维度
- **按日期分组**: 统计每天的数据量

## 数据导入导出功能

### 1. 数据导出功能

#### 功能描述
支持将表格数据导出为CSV或Excel文件。

#### 单表导出(CSV)
1. 在数据管理页面,点击"导出数据"按钮
2. 系统生成CSV文件
3. 文件名格式: {表格名称}_export.csv
4. 自动下载CSV文件
5. CSV格式:
   - UTF-8编码
   - Windows换行符(\r\n)
   - 包含表头和数据行
   - 特殊字符处理(逗号、引号、换行符)

#### 全量导出(Excel)
1. 在设置菜单中,点击"导出所有数据"按钮
2. 系统生成Excel文件
3. 文件名: all_data.xlsx
4. 自动下载Excel文件
5. Excel格式:
   - 每个表格一个工作表
   - 每个表格包含两个工作表:数据工作表和属性工作表
   - 数据工作表: 包含所有数据和创建时间
   - 属性工作表: 包含列配置(column_name, data_type, dropDown, autoIncrement, prefix, hidden)
   - 支持中文文件名和工作表名

#### 权限控制
- 单表导出: 管理员和组长
- 全量导出: 仅管理员

---

### 2. 数据导入功能

#### 功能描述
支持从CSV或Excel文件导入数据。

#### 单表导入(CSV)
1. 在数据管理页面,点击"导入数据"按钮
2. 弹出导入数据模态框
3. 上传CSV文件或粘贴CSV内容
4. 点击"导入"
5. 系统验证:
   - CSV表头必须与表格列配置匹配
   - 列数量必须一致
   - 列名称必须一致
6. 验证成功:
   - 导入数据
   - 显示导入结果(成功数、失败数、错误信息)
7. 刷新数据列表

#### 全量导入(Excel)
1. 在设置菜单中,点击"导入数据"按钮
2. 弹出导入数据模态框
3. 上传Excel文件
4. 点击"导入"
5. 系统处理:
   - 读取Excel文件
   - 识别数据工作表和属性工作表
   - 创建或更新表格结构
   - 导入数据
   - 更新自增序列
6. 显示导入结果(每个表格的导入状态)
7. 刷新表格列表和数据

#### Excel文件格式要求
- 文件扩展名: .xlsx或.xls
- 数据工作表: 包含表头和数据行,最后一列为"创建时间"
- 属性工作表(可选): 包含列配置,表头为[column_name, data_type, dropDown, autoIncrement, prefix, hidden]
- 工作表名称: 数据工作表名称为表格名称,属性工作表名称为"{表格名称}_属性"

#### 权限控制
- 单表导入: 管理员和组长
- 全量导入: 仅管理员

## 数据模型与存储

### 1. 数据库模型

#### User(用户表)
```python
{
    id: Integer (主键)
    username: String(50, 唯一, 非空)
    password: String(255, 非空)
    role: String(20, 非空)  # admin/leader/member
    first_login: Boolean (默认True)
    created_at: DateTime
    updated_at: DateTime
}
```

#### TableStructure(表格结构表)
```python
{
    id: Integer (主键)
    table_name: String(255, 非空)
    columns: Text (非空, JSON格式)
    created_at: DateTime
    updated_at: DateTime
}
```

#### InventoryData(库存数据表)
```python
{
    id: Integer (主键)
    table_id: Integer (外键, 非空)
    data: Text (非空, JSON格式)
    created_by: Integer (外键, 非空)
    created_at: DateTime
    updated_at: DateTime
}
```

#### OperationLog(操作日志表)
```python
{
    id: Integer (主键)
    user_id: Integer (外键, 非空)
    operation: String(255, 非空)
    table_id: Integer (可选)
    data_id: Integer (可选)
    created_at: DateTime
}
```

#### AutoIncrementSequence(自增序列表)
```python
{
    id: Integer (主键)
    table_id: Integer (外键, 非空)
    column_name: String(255, 非空)
    current_value: Integer (默认0, 非空)
    created_at: DateTime
    updated_at: DateTime
    唯一约束: (table_id, column_name)
}
```

#### BugReport(Bug反馈表)
```python
{
    id: Integer (主键)
    title: String(255, 非空)
    content: Text (非空)
    reporter_username: String(50, 非空)
    is_resolved: Boolean (默认False)
    created_at: DateTime
    updated_at: DateTime
}
```

### 3. API接口

#### 认证相关
- `POST /api/v1/auth/login`: 登录
- `GET /api/v1/auth/me`: 获取当前用户信息
- `POST /api/v1/auth/reset-password`: 重置密码
- `POST /api/v1/auth/change-password`: 修改密码

#### 用户管理
- `GET /api/v1/users`: 获取用户列表
- `POST /api/v1/users`: 创建用户
- `PUT /api/v1/users/<user_id>`: 更新用户
- `DELETE /api/v1/users/<user_id>`: 删除用户

#### 表格管理
- `GET /api/v1/tables`: 获取表格列表
- `POST /api/v1/tables`: 创建表格
- `GET /api/v1/tables/<table_id>`: 获取表格详情
- `PUT /api/v1/tables/<table_id>`: 更新表格
- `DELETE /api/v1/tables/<table_id>`: 删除表格
- `DELETE /api/v1/tables/all`: 删除所有表格
- `POST /api/v1/tables/<table_id>/check-auto-increment`: 检查自增列

#### 数据管理
- `GET /api/v1/tables/<table_id>/data`: 获取数据列表
- `POST /api/v1/tables/<table_id>/data`: 添加数据
- `GET /api/v1/tables/<table_id>/data/<data_id>`: 获取数据详情
- `PUT /api/v1/tables/<table_id>/data/<data_id>`: 更新数据
- `DELETE /api/v1/tables/<table_id>/data/<data_id>`: 删除数据
- `DELETE /api/v1/tables/<table_id>/data`: 批量删除数据
- `POST /api/v1/tables/<table_id>/data/batch`: 批量添加数据

#### 报表统计
- `GET /api/v1/reports/<table_id>/stats`: 获取统计数据

#### 数据导入导出
- `GET /api/v1/tables/<table_id>/export`: 导出单表数据(CSV)
- `POST /api/v1/tables/<table_id>/import`: 导入单表数据(CSV)
- `GET /api/v1/export-all-data`: 导出所有数据(Excel)
- `POST /api/v1/import-data`: 导入所有数据(Excel)

#### Bug反馈
- `POST /api/v1/bugs`: 提交Bug
- `GET /api/v1/bugs`: 获取Bug列表
- `PUT /api/v1/bugs/<bug_id>`: 更新Bug状态
- `DELETE /api/v1/bugs/<bug_id>`: 删除Bug
